# dotfiles

## alacritty.toml

### copy to the config dir and create a symbolic link
```
ln -s ~/dotfiles/alacritty/alacritty.toml ~/.config/alacritty.toml
```

### .vimrc

### copy to the root dir and create a symbolic link
```
ln -s ~/dotfiles/.vimrc ~/
```

### .bashrc

### copy to the root dir and create a symbolic link
```
ln -s ~/dotfiles/.bashrc ~/
```
